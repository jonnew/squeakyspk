Win32 Port of Autoclass 3.2.1

Gimme something that works.

Notes

  • Tabs in input are not handled gracefully
  • Make sure you don't have unix-format text files (i.e., CRLF, not LF)

Binary Download

  • Everything in this directory, or
  • Autoclass-3-2-1.win32.bin.zip

    • Other Stuff you will probably need in order to run it

    • Unix utilities for Win32 (there are free GNU versions which I don't have handy)
      • mv.exe
      • rm.exe
    • The Microsoft DLLs in this directory, including
      • MFC42D.DLL
      • MSVCIRTD.DLL
      • MSVCP50D.DLL
      • MSVCRTD.DLL

    Gonna Build it yourself?

    Watch Out!

    • When patching the original source, you _must_ specify the "-l" option.
    • Although safe_file_writing_p defaults off for Win32, it could be used if you provide mv and rm.  In that case, their absence would not be handled gracefully.

    Tools Required

    • Microsoft Visual C++ 5.0
    • patch.exe [port of a Unix utility that applies a patch file (the output of diff) to your code.

    Source

    • The Win32 specific stuff, patch.zip
      • autoclass.win32.patch
      • both AutoclassC.dsp and AutoclassC.dsw, or
      • autoclass.make.win32.msvc
    • The Autoclass-C distribution, either
      • autoclass-c-3-2-1.tar.gz, or
      • autoclass-c-3-2-1.zip

    What you will have to do

    • unpack everything
    • patch the Autoclass source distribution with "patch < win32.patch"
    • hopfully just run "nmake /f autoclass.make.win32.msvc"
    • alternatively load the .dsw file and choose "build".
    • have good luck!
